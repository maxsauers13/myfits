## How to Run

1. Frontend
- cd to client
- run "npm start"
- open localhost:3000/home

2. Middleware
- cd to server
- run "node index.js"
- (will have to set up MySql DB separately)

3. Backend
- cd to server/flask/api.py
- run "python api.py"
